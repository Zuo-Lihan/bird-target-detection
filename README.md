# 鸟类目标检测

#### 介绍
鸟类目标检测实践

#### 软件架构
anaconda 
keras
YOLO算法


#### 文件结构说明

在 **ImageSets/Main** 下都先建立一个对应类别的鸟类的文件，再在该文件内部生成> "train.txt"、> "trainval.txt"、> "val.txt"、> "test.txt"，有利于项目同时实现单目标检测和多目标检测，让项目结构更加清晰。
